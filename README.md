# By Ellen Interiors

Website voor **By Ellen Interiors** — interieurontwerp, advies & styling in Zuid-Holland.

Statische site (HTML/CSS/JS), geen build-stap nodig. Open `index.html` of host via GitHub Pages.

## Stijl
- Palet "Olijf & linnen": gebroken wit `#F7F4EE`, greige `#DAD3C5`, steen `#9E978A`, tekst `#3B3B3B`, pop-kleur **olijf `#6C6E36`**.
- Display-lettertype: Playfair Display · tekst: Manrope · logo: `assets/logo-Ellen.png`.
- Opening: olijf scherm met het logo dat zacht infadet en in het midden opensplitst.

## Structuur
- `index.html` — homepage
- `css/style.css` — stijl
- `js/main.js` — opening, navigatie, scroll-reveals
- `img/` — web-geoptimaliseerde projectfoto's
- `assets/` — logo

Pagina's Over, Portfolio, Werkwijze, Klanten en Contact volgen in dezelfde stijl.
